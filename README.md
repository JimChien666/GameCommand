# 利用 Laravel Commands 寫一個遊戲專區
```
    ______                   
   / ____/___ _____ ___  ___ 
  / / __/ __ `/ __ `__ \/ _ \
 / /_/ / /_/ / / / / / /  __/
 \____/\__,_/_/ /_/ /_/\___/ 
 ```
### 遊戲指令介紹
1. 查看遊戲清單:
- game:list
2. 查看遊戲玩法:
- game:rule '<遊戲名稱>'
3. 開始遊戲:
- game:play '<遊戲名稱>'

### 遊戲指令介紹
1. 查看遊戲清單:
- game:list
2. 查看遊戲玩法:
- game:rule '<遊戲名稱>'
3. 開始遊戲:
- game:play '<遊戲名稱>'

### 目前的遊戲列表

1. 終極密碼(UltimatePassword)

### 終極密碼
遊戲規則如下
- 可選擇玩家人數，場數
- 隨機選擇起始玩家，之後依場次更換起始玩家
- 規則如終極密碼規則，猜到數字的人獲勝
- 先到獲勝場數的玩家獲勝
